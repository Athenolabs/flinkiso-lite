How to Install

Requirements: 

Basic knowledge of server administration (IIS/Apache) & php.

Server Requirements:

We prefer XAMPP (http://www.apachefriends.org/index.html)

  1. Apache server with MySQL
    1. If you have apache server make user you have
      1. PHP installed and working
      2. mod_rewrite is ON
    2. IIS server with MySQL
      1. IIS6+

      2. To install PHP on IIS visit : http://php.iis.net/

      3. mod_rewrite : To enable mod_rewrite on IIS visit :
        http://www.iis.net/learn/extensions/url-rewrite-module/importing-apache-modrewrite-rules

      4. MySQL : To install MySQL on windows, visit :
        http://dev.mysql.com/downloads/windows/

    3. XAMPP on windows: If you do not have any of the following, you can download XAMPP for either windows or mac or Linux and install. To download XAMPP visit:
      http://www.apachefriends.org/download.html . XAMPP will install all the required tools on your machine or server.

Installation:

  1. Download & copy "FlinkISO_0.0.1.tar.gz" into your server's root folder.
  2. Open (unzip) the file.
  3. Name your folder e.g "flinkiso"
  4. Make sure that "flinkiso" folder has write permissions (recursive).
  5. Open your favorite browser and enter the following in the address bar: http://localhost/flinkiso/
  6. Installer will begin installing the application.
  7. During the installation, the installer will ask for information related to the database.
  8. Enter your database connection information, e.g:
    1. Host: localhost
    2. Database name: FlinkISO or any name you choose to have
    3. User: root
    4. Prefix: flink_
    5. Password: Your password
  9. Click Submit.
  10. On submit, the installer will install and create required files / folders & database.
  11. After successful installation, system will redirect you to register your copy of FlinkISO. Register your copy and start using FlinkISO for free.

Product License Key

When you download FlinkISO from official website (www.flinkiso.com) you will receive an email along with your License Key.

Enter your license key from the email and FlinkISO will automatically add your required details to the form and you will be prompted to the next step.

After successful registration, you can access FlinkISO official forum, help & support

Do not share this key with anyone.

  2. MR meetings information
